Introduction

Swagmail is an Email & SMS Marketing application for any kind of business marketing purpose. You can get rid of expensive email services like Mailchimp, ActiveCampaign. Swagmail is also built for saas business since Swagmail integrates easily with payment gateways like Paypal, Stripe and it offers all the needed tools to create pricing plans.

You can send an unlimited number of email campaigns (newsletters) to an unlimited number of lists containing an unlimited number of subscribers.

You can use multiple SMTP or PHP mail at the same time with Gmail, Yahoo, Mailgun, Zoho, Elasticmail. If you would like to send emails from your own SMTP server.
